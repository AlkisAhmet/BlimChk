# BlimChk
Blim account checker is an open-source account checker
For Blim. Made with python.

## Instalation

```
apt update && apt upgrade
pkg install python2
pkg install git
Git clone https://github.com/Mr-Z3r0/BlimChk.git
cd blimchk
python2 blim.py
```
